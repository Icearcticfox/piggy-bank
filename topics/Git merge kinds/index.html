    1. Fast-forward Merge (–°–ª–∏—è–Ω–∏–µ —Å –ø–µ—Ä–µ–º–æ—Ç–∫–æ–π)

    –≠—Ç–æ –ø—Ä–æ—Å—Ç–æ–µ –ø–µ—Ä–µ–º–µ—â–µ–Ω–∏–µ —É–∫–∞–∑–∞—Ç–µ–ª—è —Ü–µ–ª–µ–≤–æ–π –≤–µ—Ç–∫–∏ –Ω–∞ –ø–æ—Å–ª–µ–¥–Ω–∏–π –∫–æ–º–º–∏—Ç –∏—Å—Ö–æ–¥–Ω–æ–π –≤–µ—Ç–∫–∏, –µ—Å–ª–∏ –∏—Å—Ç–æ—Ä–∏—è –ª–∏–Ω–µ–π–Ω–∞—è –∏ —Ü–µ–ª–µ–≤–∞—è –≤–µ—Ç–∫–∞ –Ω–µ –∏–º–µ–µ—Ç –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—ã—Ö –∫–æ–º–º–∏—Ç–æ–≤. –ö–∞–∫ –æ–±—Å—É–∂–¥–∞–ª–æ—Å—å —Ä–∞–Ω–µ–µ, –Ω–æ–≤—ã–π –∫–æ–º–º–∏—Ç –Ω–µ —Å–æ–∑–¥–∞—ë—Ç—Å—è.

    2. No Fast-forward Merge (–°–ª–∏—è–Ω–∏–µ –±–µ–∑ –ø–µ—Ä–µ–º–æ—Ç–∫–∏)

    Git —Å–æ–∑–¥–∞—ë—Ç –æ—Ç–¥–µ–ª—å–Ω—ã–π –∫–æ–º–º–∏—Ç —Å–ª–∏—è–Ω–∏—è, –¥–∞–∂–µ –µ—Å–ª–∏ –∏—Å—Ç–æ—Ä–∏—è –ø–æ–∑–≤–æ–ª—è–µ—Ç –ª–∏–Ω–µ–π–Ω–æ–µ —Å–ª–∏—è–Ω–∏–µ. –≠—Ç–æ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è —è–≤–Ω–æ–≥–æ —Ñ–∏–∫—Å–∞—Ü–∏–∏ –º–æ–º–µ–Ω—Ç–∞ —Å–ª–∏—è–Ω–∏—è, —á—Ç–æ –ø–æ–º–æ–≥–∞–µ—Ç –≤ –∞–Ω–∞–ª–∏–∑–µ –∏—Å—Ç–æ—Ä–∏–∏.

    3. Squash Merge (–°–ª–∏—è–Ω–∏–µ —Å "—Ä–∞–∑–¥–∞–≤–ª–∏–≤–∞–Ω–∏–µ–º" –∫–æ–º–º–∏—Ç–æ–≤)

    Squash merge –æ–±—ä–µ–¥–∏–Ω—è–µ—Ç –≤—Å–µ –∫–æ–º–º–∏—Ç—ã –∏–∑ –∏—Å—Ö–æ–¥–Ω–æ–π –≤–µ—Ç–∫–∏ –≤ –æ–¥–∏–Ω –µ–¥–∏–Ω—ã–π –∫–æ–º–º–∏—Ç –≤ —Ü–µ–ª–µ–≤–æ–π –≤–µ—Ç–∫–µ. –≠—Ç–æ –ø–æ–ª–µ–∑–Ω–æ, –µ—Å–ª–∏ –≤—ã —Ö–æ—Ç–∏—Ç–µ —Å–æ—Ö—Ä–∞–Ω–∏—Ç—å —á–∏—Å—Ç—É—é –∏ –ø–æ–Ω—è—Ç–Ω—É—é –∏—Å—Ç–æ—Ä–∏—é –±–µ–∑ –º–Ω–æ–∂–µ—Å—Ç–≤–∞ –º–µ–ª–∫–∏—Ö –∫–æ–º–º–∏—Ç–æ–≤, —Å–¥–µ–ª–∞–Ω–Ω—ã—Ö –≤ –ø—Ä–æ—Ü–µ—Å—Å–µ —Ä–∞–±–æ—Ç—ã –Ω–∞–¥ —Ñ–∏—á–µ–π. –ü—Ä–∏ —ç—Ç–æ–º –∏—Å—Ö–æ–¥–Ω—ã–µ –∫–æ–º–º–∏—Ç—ã –Ω–∞ –≤–µ—Ç–∫–µ –Ω–µ —Å–æ—Ö—Ä–∞–Ω—è—é—Ç—Å—è, –Ω–æ –≤–µ—Å—å –∏—Ö –∫–æ–¥–æ–≤—ã–π –≤–∫–ª–∞–¥ —Å–æ–¥–µ—Ä–∂–∏—Ç—Å—è –≤ –æ–¥–Ω–æ–º –Ω–æ–≤–æ–º –∫–æ–º–º–∏—Ç–µ.

    –ü—Ä–∏–º–µ—Ä: 1. –ò—Å—Ç–æ—Ä–∏—è –∏—Å—Ö–æ–¥–Ω–æ–π –≤–µ—Ç–∫–∏: A -- B -- C (main) \ D -- E -- F (feature) 2. –ü–æ—Å–ª–µ squash-—Å–ª–∏—è–Ω–∏—è –≤—Å–µ –∫–æ–º–º–∏—Ç—ã –∏–∑ feature –æ–±—ä–µ–¥–∏–Ω—è—é—Ç—Å—è –≤ –æ–¥–∏–Ω: A -- B -- C -- G (main)

    –ö–æ–º–º–∏—Ç G —Å–æ–¥–µ—Ä–∂–∏—Ç –∏–∑–º–µ–Ω–µ–Ω–∏—è –∏–∑ D, E –∏ F.

    –î–ª—è –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è squash merge –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –∫–æ–º–∞–Ω–¥–∞:

    –ü–æ—Å–ª–µ —ç—Ç–æ–≥–æ –≤—ã –¥–æ–ª–∂–Ω—ã –≤—Ä—É—á–Ω—É—é –∑–∞—Ñ–∏–∫—Å–∏—Ä–æ–≤–∞—Ç—å —Å–ª–∏—è–Ω–∏–µ —Å –ø–æ–º–æ—â—å—é git commit.

    4. Rebase Merge (–ü–µ—Ä–µ–±–∞–∑–∏—Ä–æ–≤–∞–Ω–∏–µ)

    Rebase –≤–º–µ—Å—Ç–æ –æ–±—ã—á–Ω–æ–≥–æ —Å–ª–∏—è–Ω–∏—è "–ø–µ—Ä–µ–ø–∏—Å—ã–≤–∞–µ—Ç" –∏—Å—Ç–æ—Ä–∏—é –∏–∑–º–µ–Ω–µ–Ω–∏–π –≤–µ—Ç–∫–∏, –∫–∞–∫ –µ—Å–ª–∏ –±—ã –≤—ã –Ω–∞—á–∏–Ω–∞–ª–∏ –≤–µ—Ç–∫—É —Å –ø–æ—Å–ª–µ–¥–Ω–µ–≥–æ –∫–æ–º–º–∏—Ç–∞ —Ü–µ–ª–µ–≤–æ–π –≤–µ—Ç–∫–∏. –≠—Ç–æ –ø–æ–∑–≤–æ–ª—è–µ—Ç —Å–æ—Ö—Ä–∞–Ω–∏—Ç—å –ª–∏–Ω–µ–π–Ω—É—é –∏—Å—Ç–æ—Ä–∏—é –±–µ–∑ —Å–æ–∑–¥–∞–Ω–∏—è –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—ã—Ö –∫–æ–º–º–∏—Ç–æ–≤ —Å–ª–∏—è–Ω–∏—è.

    –ü—Ä–∏–º–µ—Ä: 1. –í—ã –Ω–∞—á–∏–Ω–∞–µ—Ç–µ –≤–µ—Ç–∫—É feature –æ—Ç C –∏ –¥–µ–ª–∞–µ—Ç–µ –Ω–µ—Å–∫–æ–ª—å–∫–æ –∫–æ–º–º–∏—Ç–æ–≤: A -- B -- C (main) \ D -- E -- F (feature) 2. –ù–∞ main –ø—Ä–æ–∏—Å—Ö–æ–¥—è—Ç –Ω–æ–≤—ã–µ –∫–æ–º–º–∏—Ç—ã: A -- B -- C -- G -- H (main) \ D -- E -- F (feature) 3. –ü—Ä–∏ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–∏ rebase –≤–µ—Ç–∫–∞ feature "–ø–µ—Ä–µ–±–∞–∑–∏—Ä—É–µ—Ç—Å—è" –Ω–∞ –ø–æ—Å–ª–µ–¥–Ω–∏–π –∫–æ–º–º–∏—Ç –≤ main: A -- B -- C -- G -- H -- D' -- E' -- F' (main/feature)

    –≠—Ç–æ –ø–æ–∑–≤–æ–ª—è–µ—Ç –∏–∑–±–µ–∂–∞—Ç—å —Å–ª–æ–∂–Ω—ã—Ö –≥—Ä–∞—Ñ–æ–≤ –∏ –≤–µ—Ç–≤–ª–µ–Ω–∏–π, –Ω–æ –º–æ–∂–µ—Ç —É—Å–ª–æ–∂–Ω–∏—Ç—å —Ä–∞–∑—Ä–µ—à–µ–Ω–∏–µ –∫–æ–Ω—Ñ–ª–∏–∫—Ç–æ–≤. –ö–æ–º–∞–Ω–¥–∞ –¥–ª—è —Ä–µ–±–µ–π—Å–∞:

    5. Three-way Merge (–¢—Ä—ë—Ö—Å—Ç–æ—Ä–æ–Ω–Ω–µ–µ —Å–ª–∏—è–Ω–∏–µ)

    –¢—Ä—ë—Ö—Å—Ç–æ—Ä–æ–Ω–Ω–µ–µ —Å–ª–∏—è–Ω–∏–µ –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç, –∫–æ–≥–¥–∞ –≤ –æ–±–µ–∏—Ö –≤–µ—Ç–∫–∞—Ö –±—ã–ª–∏ –∫–æ–º–º–∏—Ç—ã –ø–æ—Å–ª–µ —Ä–∞–∑–¥–µ–ª–µ–Ω–∏—è –≤–µ—Ç–∫–∏. –í —Ç–∞–∫–æ–º —Å–ª—É—á–∞–µ Git –∏—Å–ø–æ–ª—å–∑—É–µ—Ç "–æ–±—â–∏–π –ø—Ä–µ–¥–æ–∫" (common ancestor) –¥–≤—É—Ö –≤–µ—Ç–æ–∫ –∏ –≤—ã–ø–æ–ª–Ω—è–µ—Ç —Å–ª–∏—è–Ω–∏–µ, —Ä–∞–∑—Ä–µ—à–∞—è –∫–æ–Ω—Ñ–ª–∏–∫—Ç—ã –º–µ–∂–¥—É –∏–∑–º–µ–Ω–µ–Ω–∏—è–º–∏ –≤ –≤–µ—Ç–∫–∞—Ö. –≠—Ç–æ –æ–±—ã—á–Ω—ã–π –ø—Ä–æ—Ü–µ—Å—Å, –∫–æ–≥–¥–∞ fast-forward —Å–ª–∏—è–Ω–∏–µ –Ω–µ–≤–æ–∑–º–æ–∂–Ω–æ, –∏ Git —Å–æ–∑–¥–∞—ë—Ç –Ω–æ–≤—ã–π –∫–æ–º–º–∏—Ç —Å–ª–∏—è–Ω–∏—è.

    –ü—Ä–∏–º–µ—Ä: 1. –ò—Å—Ç–æ—Ä–∏—è –¥–≤—É—Ö –≤–µ—Ç–æ–∫: A -- B -- C (–æ–±—â–∏–π –ø—Ä–µ–¥–æ–∫) | \ | D -- E (feature) \ F -- G (main)

    –ü—Ä–∏ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–∏ —Ç—Ä—ë—Ö—Å—Ç–æ—Ä–æ–Ω–Ω–µ–≥–æ —Å–ª–∏—è–Ω–∏—è, Git –±–µ—Ä—ë—Ç –∫–æ–º–º–∏—Ç—ã C, G, –∏ E, —á—Ç–æ–±—ã —Å–≥–µ–Ω–µ—Ä–∏—Ä–æ–≤–∞—Ç—å –Ω–æ–≤—ã–π –∫–æ–º–º–∏—Ç M, –∫–æ—Ç–æ—Ä—ã–π –≤–∫–ª—é—á–∞–µ—Ç –≤—Å–µ –∏–∑–º–µ–Ω–µ–Ω–∏—è: A -- B -- C -- F -- G -- M (main) \ / D -- E

    6. Octopus Merge (–û—Å—å–º–∏–Ω–æ–∂—å–µ —Å–ª–∏—è–Ω–∏–µ)

    –ü—Ä–∏–º–µ—Ä —Å–ª–∏—è–Ω–∏—è —Ç—Ä—ë—Ö –≤–µ—Ç–æ–∫:

    –ö–æ–≥–¥–∞ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –∫–∞–∂–¥—ã–π —Å–ø–æ—Å–æ–±?
